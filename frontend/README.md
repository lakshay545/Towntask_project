# Towntask Frontend

This is the frontend for the Location-Based Local Skill & Work Matching Platform, built with React, Vite, and Tailwind CSS.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Run the development server:**
    ```bash
    npm run dev
    ```
3.  **Build for production:**
    ```bash
    npm run build
    ```
